JavaScript Example 
// Initialize the Crazy-Gary API client
const crazyGary = new CrazyGaryAPI({
  apiKey: 'your-api-key-here',
  baseURL: 'https://api.crazy-gary.com'
});

// Generate code using AI
const code = await crazyGary.generateCode({
  prompt: 'Create a React component for a todo list',
  language: 'javascript',
  framework: 'react'
});

console.log(code);

API Overview

AI-Powered

Advanced AI capabilities powered by Harmony integration for intelligent code generation and analysis.

MCP Tools

Model Context Protocol (MCP) integration for extended functionality and tool orchestration.

Real-time Monitoring

Comprehensive monitoring and observability features with WebSocket support for real-time updates.

Enterprise Security

Advanced security features including rate limiting, input sanitization, and comprehensive authentication.

Authentication

The Crazy-Gary API uses JWT-based authentication. Include your API key in the Authorization header of every request.

Authorization Header 
Authorization: Bearer your-jwt-token-here
Content-Type: application/json

JWT Tokens

Use JWT tokens obtained from the authentication endpoint

API Keys

Direct API key authentication for server-to-server communication

API Endpoints

Authentication

POST /api/auth/login User authentication
POST /api/auth/register User registration
POST /api/auth/refresh Token refresh

Code Generation

POST /api/generate AI code generation
POST /api/analyze Code analysis
POST /api/optimize Code optimization

MCP Tools

GET /api/tools List available tools
POST /api/tools/execute Execute tool
GET /api/tools/status Tool status

Monitoring

GET /api/monitoring/metrics System metrics
GET /api/monitoring/logs System logs
GET /api/monitoring/health Health status
